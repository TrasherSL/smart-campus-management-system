Smart Campus Management System API

Server is running! The API is available and ready to accept requests.

API Information

This is the backend API for the Smart Campus Management System. It provides endpoints for user authentication, user management, analytics, and dashboard data.

Version: 1.0.0

Environment: Development

Available Endpoints

GET /health

Check the health status of the API server.

POST /api/auth/login

Authenticate a user and get a token.

POST /api/auth/register

Register a new user.

GET /api/auth/logout

Log out a user (requires authentication).

GET /api/users

Get a list of users (requires authentication).

GET /api/analytics

Get analytics data (requires authentication).

GET /api/dashboard

Get dashboard data (requires authentication).
